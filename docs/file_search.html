file_search.main()¶
file_search.print_entries(pattern, minSize, maxSize, case)¶

Select all database entries matching the pattern. Print the entries’ path, size, file type and last modification date.

Parameters

  • pattern (string) – pattern for matching the entries

  • minSize (int) – only match entries this size or bigger, default 0 bytes

  • maxSize (int) – only match entries up to this size, default 9000000000000000000 bytes

  • case (bool) – sets case-sensitive pattern matching
