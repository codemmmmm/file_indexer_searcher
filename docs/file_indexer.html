file_indexer module¶

Scans and saves all directory entries in a linux filesystem into a SQLite DB, except “/proc”, “/var”, “/run”, “/sys”, “/dev” and “/boot”. Configuration data is saved in “/var/lib/file_index_search/config.yaml”.
